\# internet-tehnologije-2025-rentabike_2021_0076




Ovo je aplikacija za iznajmljivanje bicikala, trotineta i rolera. Projekat sadrži dva dela:

\- \*\*Backend\*\*: Laravel API

\- \*\*Frontend\*\*: React aplikacija



\## Struktura projekta

projekat/

├── backend/ # Laravel API

├── frontend/ # React aplikacija

└── README.md 

\## Backend (Laravel)

\*\*Tehnologije:\*\* PHP 8, Laravel 11, MySQL  



\### Instalacija

Backend

composer install

php artisan migrate

php artisan serve

Frontend

npm install

npm start
